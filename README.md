#  10連戦オセロ (Python/Flask)

YouTube動画で解説した「戦略AI」を搭載したWeb版オセロです。
AIとの10連戦を戦い抜きましょう！

## 📺 動画解説はこちら
https://www.youtube.com/watch?v=Romwjzs_ow8
※制作の裏側や、AIの重み付けアルゴリズムについて詳しく解説しています。

## 🌟 主な機能
- **戦略的AI**: 「角」を狙い「角の隣」を避ける評価関数（重み付けマップ）を搭載。
- **10連戦カウント**: セッション管理による戦績の累積表示。

## 🛠 使用技術
- **Backend**: Python 3.x / Flask
- **Frontend**: HTML5 /  JavaScript
- **Design**: Canva (Slide & Thumbnails)

## 🚀 実行方法

1. リポジトリをクローンまたはダウンロードします。
2. 必要なライブラリをインストールします。
```bash
pip install flask
```
3. アプリケーションを起動します。
```bash
python3 app.py
```
4. ブラウザで http://127.0.0.1:5000 にアクセスしてください。

## 🧠 AIの重み付けマップ本アプリのAIは、以下のスコア表（評価関数）に基づいて、最も期待値の高いマスを選択します。

マスの種類         スコア    理由
四隅 (Corner)      +30      奪われない最強の拠点
端 (Edge)          +5       辺の確保は安定につながる
角の隣 (X/C-move)  -15      相手に角を取られるリスクが高い
